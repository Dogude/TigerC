# TigerC-Compiler
TigerC Compiler - In memory of Dennis Ritchie
Minimal C compiler experiment written from scratch.

## Current Features
- Tokenizer
- Expression parser
- Shunting Yard algorithm
- Bytecode evaluator

## Roadmap
- C23 keywords
- PE32+ EXE generation
- Minimal runtime
